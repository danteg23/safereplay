# SafeReplay approved v2 visual contract

The paired source-of-truth images are:

- `accepted-desktop-matches.png`: desktop shell, navigation density, date/filter rails,
  open fixture rows, spacing, typography, and overall white premium product character.
- `accepted-mobile-detail.png`: mobile match header, format hierarchy, quiet alternative
  links, outline play controls, and the First half / Second half treatment.

User decisions after the screenshots:

- Remove the Watched destination and watched-history feature.
- Do not show availability/status dots or availability copy anywhere in Matches.
- Matches never have play buttons. Available rows open detail; unavailable rows are
  washed out and cannot be activated.
- Time zone is changed only from Settings. Manila and Oslo are pinned above the full,
  searchable list.
- Alternatives are text links, never separate play buttons.
- Full match has no general play action; only First half and Second half do.
- Desktop follows the same Matches → detail navigation as mobile.
- Team emblems are optional only in the detail header, using real mapped assets and only
  when they improve the composition.
- The only play control allowed in Matches is a live shortcut shown from kickoff through
  a three-hour live window. It opens the configured primary live directory. Detail may
  show two quiet live-directory alternatives. All live UI disappears outside that window.
